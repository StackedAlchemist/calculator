# 🧮 Arithmetica — Scientific Calculator

> *"Where numbers bend to the will of the alchemist."*

A full scientific calculator with PEMDAS-compliant expression parsing, trigonometric functions, logarithms, factorials, mathematical constants, and an expression history log.

**[🔮 Open Calculator](https://stackedalchemist.github.io/stacked-alchemist/calculator/)**

---

## ✦ Features

- **PEMDAS Expression Parser** — Evaluates full mathematical expressions respecting order of operations
- **Scientific Functions** — sin, cos, tan, asin, acos, atan (degrees and radians)
- **Logarithms** — log (base 10) and ln (natural log)
- **Factorials** — Integer factorial calculation (`n!`)
- **Constants** — π (pi) and e (Euler's number)
- **Expression History** — Scrollable log of previous calculations; click any to reuse
- **Keyboard Support** — Full number pad and operator keyboard input
- **Error Handling** — Graceful handling of undefined operations (divide by zero, invalid input)

## 🎹 Keyboard Support

| Key | Action |
|---|---|
| `0–9` | Number input |
| `+ - * /` | Operators |
| `Enter` or `=` | Evaluate |
| `Escape` or `C` | Clear |
| `Backspace` | Delete last character |
| `.` | Decimal point |
